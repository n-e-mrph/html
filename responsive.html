Responsive Web Design



What's Responsive Design?


Responsive design is a modern approach in web development that makes a website function well on many screen sizes - from your phone to your desktop. In the early oughts, web developers used to have to make multiple sites if they wanted the webpage to be viewable on more than one type of screen. Now designers can make websites that automatically adjust to different screen sizes.


Fluid grids and flexible images are a couple tools that developers use to make design responsive. Percentages are used, instead of measurements, so that the elements can resize based on the parameters of the screen. Media queries are another tool used for responsive design. With media queries, developers can make separate sets of rules for each device, in one codebase.


The invention of responsive design has been revolutionary in the history of web design. When the iPhone first was released there were few websites that were mobile friendly. Now, with responsive design, the user experience has improved as webpages are simpler to navigate, and text is easier to read.
